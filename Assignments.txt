Run below commands moving to rotc folder on terminal

-- Manual

1. Start an nginx container with port forwarding to local & check if the application is up
docker run -p 8080:80 --name myweb -d nginx
curl localhost:8080

2. Check logs
docker logs myweb

3. Go inside the container
docker exec -it myweb sh

4. Stop the container
docker stop myweb

-- Automated
1. Start an nginx container
docker-compose up -d

2. Check if the application is up
in browser, give localhost:8080
